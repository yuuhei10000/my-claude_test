# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Claude Code の検証用プロジェクト。Node.js で動作するシンプルなコンソール出力アプリケーション。

## Commands

- `node main.js` - アプリケーション実行

## Architecture

単一ファイル構成（`main.js`）。エントリポイントでコンソール出力を行い、`helloWorld` 関数を定義・実行する。
